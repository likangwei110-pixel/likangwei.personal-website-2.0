# Kangwei Li (李康炜) - Personal Portfolio

A minimalist, editorial-style personal portfolio website inspired by `calebixca.com`. Designed for Product Manager candidates, featuring trilingual support (CN/EN/DE) and a custom inline CMS.

## ✨ Features

- **High-Contrast Design**: Minimalist aesthetic with a bold orange/coral accent.
- **Trilingual Core**: Seamless switching between Chinese, English, and German.
- **Inline Editing**: Live CMS mode where text can be edited directly on the page (persisted via LocalStorage).
- **Curated Gallery**: Horizontal scrolling photography gallery with trilingual metadata editing.
- **Responsive Layout**: Precision-crafted for both desktop and mobile viewing.
- **Tech Stack**: React 19, Vite, Tailwind CSS 4, Framer Motion, Lucide Icons.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm / yarn

### Installation

1. Clone the repository
   ```bash
   git clone <your-repo-url>
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server
   ```bash
   npm run dev
   ```

## 🛠 Project Structure

- `src/App.tsx`: Main application logic and UI.
- `src/types.ts`: Trilingual translation dictionary and data schemas.
- `src/index.css`: Global styles and Tailwind 4 configurations.
